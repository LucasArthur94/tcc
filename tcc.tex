% ========== Resumo ==========
\begin{resumo}
Ao longo dos anos, o processo das disciplinas de projeto de formatura do PCS ocorre de maneira extremamente manual, com entregas descentralizadas, avaliações em papel, cálculo de notas trabalhoso, entre outras características. O projeto consiste em elaborar um sistema responsável por centralizar e automatizar todo o processo, usando técnicas de Engenharia de Software, além de gerar um legado para a Escola Politécnica.
%
\\[3\baselineskip]
%
\textbf{Palavras-Chave} -- histórias de usuário, casos de uso, django, testes, validação, visão.
\end{resumo}


% ========== Abstract ==========
\begin{abstract}

Over the years, the process of the graduation project disciplines of the PCS occurs in an extremely manual way, with decentralized deliveries, paper evaluations, calculation of labor notes, among other characteristics. The project consists of elaborating a system responsible for centralizing and automating the entire process, using Software Engineering techniques, in addition to generating a legacy for the Polytechnic School.
%
\\[3\baselineskip]
%
\textbf{Keywords} -- user stories, use cases, django, tests, validation, vision.
\end{abstract}
